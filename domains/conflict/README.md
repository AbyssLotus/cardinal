# domains/conflict/

**Charter:** Vol. III Ch. 8 — Conflict. Violence is a consequence, never a feature:
every engagement traces to hostility, every hostility to events (the war nobody
declared, Ch. 8 Designer Note).

**Owns:** grievances (provenance, severity, satisfaction state); hostility state;
threats and deterrence; engagements, combat state, morale; capture, surrender, rout;
truces and terms; martial organization.

**Never owns:** the combatants' bodies (we *propose* damage; Living Systems applies it —
Ruling 9), the weapons (Physical/Economy), the reasons (other domains), the sides
(Society/Institutions), legality (Institutions), terrain (Physical) (§8.3).

**Key contracts:** no violence without a traceable cause chain; grievances are
append-only; the same resolution architecture spans tavern brawl and siege; morale can
end any engagement — extermination is never the default; no aggro radius (§8.8, §8.9).

**Appendix A:** Ruling 9 (Conflict proposes, owners apply).
