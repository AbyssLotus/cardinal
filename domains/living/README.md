# domains/living/

**Charter:** Vol. III Ch. 2 — Living Systems. Life is maintained state, never a flag;
death is the terminal state of a failed process, not a scripted event (§2.1).

**Owns:** vital state (health, energy, needs); metabolism; growth and lifecycle stage;
reproduction and inheritance; disease, injury, recovery; sensory and physical
capability; the death event.

**Never owns:** decisions (hunger is ours; going to find food is not — Ruling 4),
knowledge and memory (information layer), populations (Ecology), value (Economy),
social bonds (Society), environmental conditions (Physical) (§2.3).

**Key contracts:** needs decay deterministically and restore only through world
interaction; consumption is conservative against Physical Reality; no organism is
architecturally privileged — the protagonist starves by the same curves (§2.7, §2.8.9).

**Appendix A:** Ruling 4 (biology never decides), Ruling 12 (individuation: while
instantiated the organism is fully ours; the books balance with Ecology both ways).
