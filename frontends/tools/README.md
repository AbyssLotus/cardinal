# frontends/tools/

The historian's instruments: the inspector, causal-graph exploration, world-diff,
signature comparison dashboards (Vol. V Ch. 8). Read-only without exception — the
analytical layer that detects a dark age must be architecturally incapable of
preventing one (Vol. III Ch. 11 §11.4). World *editing* happens by editing generation
inputs and regenerating, never by writing facts into a running save (Vol. IV Ch. 4
§4.6, the Ungoverned Editor).
