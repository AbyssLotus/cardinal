# tests/probes/

Minimal worlds exercising one mechanism in isolation (Vol. IV Ch. 7 §7.3): probes
localize before references diagnose. The POC's cyberpunk/destiny/wow probes — guns,
resurrection, mana on an unmodified engine — are the founding pattern
(archive/poc-v0.1/tests/). Every exotic mechanism a package leans on gets a
minutes-scale probe world here; when the Sundered March breaks, the suspect list
starts narrow.
